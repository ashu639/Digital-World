Search Engine Optimisation

SEO is the abbreviation for search engine optimization. 'Search engine optimization (SEO) is the process of affecting the visibility of a website or a web page in a search engine's natural or organic search results.'[1] The highest ranked website that is displayed in the search results list receives a number of visitors from the search engine's users. SEO help to make sure that a site is accessible by a search engine. 'Search engines are programs that search documents for specified keywords and return a list of documents where the keywords were found.'[2] Search engines enable users to search for documents on World Wide Web. Search engines refer users to websites if they find the content, performance of a website that is the most relevant to what the user is looking for. To understand the goal of SEO, consider our internet browse habits. When we want to get some information we make use of search engines because it is the easiest and fastest way to retrieve the information. On a search query some results are displayed out of which we are more likely to go to the first page of results because they contain the relevant information about what we are looking for. As a general rule, the websites that appear on the top in the results get more users when compared to the other sites. So the goal of SEO is to get to position one by optimizing the web site so that the search engines rank the web sites as high as possible. Strategies to strengthen a web site's visibility The visibility of a site can be defined as the ease with which the users find the site among millions of the sites. Effective strategies of website depend on the knowledge of an individual business. It is necessary to follow some basic principles so that there is a pretty good chance of getting to page one. Identify and Use Keywords The first step of optimization to find required website is to make sure that we are using proper keywords in a search engine query. At first, we need to make our own list of keywords and then use the Google AdWords Keywords Tool which is designed to help in identifying the required keywords. Once the keywords are identified, make sure that the keywords appear in the body as well as metadata of the page which cannot be seen.
